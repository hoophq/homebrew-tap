# hoop.dev Homebrew tap

```sh
brew install hoophq/tap/leash
brew install hoophq/tap/hooprs
```

[Leash](https://github.com/hoophq/leash) — guardrails for AI coding agents: blocks catastrophic tool calls before they run.

[hooprs](https://github.com/hoophq/rs) — local PII and secrets risk scanner for AI coding sessions.

Everything here is published automatically at release time — casks by [GoReleaser](https://goreleaser.com) (leash), formulae by the source repo's release workflow (hooprs). Do not edit them by hand.
